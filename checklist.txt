Instalacion de un sistema de monitoreo de posiciones globales: Traccar
(en un domicilio particular).

Licencia Apache 2.0
Instalacion de tracker server en debian
opciones:
    - Instalar desde zip
    - clone Source code
        install maven 
        install jdk
        
Configuracion NAT de router para port forwarding
    - ports para el tracker server
    - ports para la interface web de seguimiento
    
Configuracion de un dns dinamico (duckdns.org)
    - alta en el proveedor del servicio gratuito
    - configuracion de un cron service para mantener la "ip publica actualizada"
    
Instalacion para prueba de una app traccar en android, tracker client.
